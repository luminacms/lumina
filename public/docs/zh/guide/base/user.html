# 人员管理

# 基础概况

lumina是一套基于组织的sass系统，角色配置里面分用户和组织类型，可将具体的权限分配到具体的人或者是组织。 社交化登录管理

接口代码事例： /oauth/{driver}/{oid} /callback/oauth/{driver}/{oid}

目前支持头条(toutiao)，微信(wechat)，github(github)，例如微信： /oauth/wechat/4306d5d5044f49cf95a666782020fecc /callback/oauth/wechat/4306d5d5044f49cf95a666782020fecc 在lumina中社交化登录已经集成到option中，你可以直接在option中去配置使用即可
